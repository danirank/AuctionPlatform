# Auction Platform

Fullstack auction platform with focus on backend quality and testability.

- Backend: ASP.NET Core Web API (.NET)  
- Frontend: React  
- Architecture: Controller → Service → Repository  
- Service layer developed with Test-Driven Development (TDD)  
- Unit testing with xUnit and Moq  
- CI pipeline with GitHub Actions  

Main focus: clean architecture, maintainable code, and production-ready backend practices.
